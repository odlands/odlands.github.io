Property Value Visualization

The Project

The property value visualization application was a month-long project for our group of three. The code is written in Java with a focus on fundamental design principles in object-oriented programming.

The Presentation

Key to the application is a user interface that is easy to navigate and pleasant to look at. Each of our main features can be accessed via tabs, each having a minimalist visual style that promote a simple and intuitive user experience. Also present is the option to view the application in light or dark mode as per the user's preference.

The Google Maps Integration

The application uses the Google Maps JavaScript API so property values can be visually represented as a heat map in any area that is read from a file. The user can also target different wards or neighborhoods to see more focused data.

The Visualizations

Three types of visualizations are included: a pie chart with ranges of assessed values, a scatter plot with neighborhoods and average property values, and a bar graph with wards and average property values.
